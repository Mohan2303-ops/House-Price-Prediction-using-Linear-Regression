# 📖 Project Overview

This project was developed as part of the **MDI3003 – Advanced Predictive Analytics** laboratory course at **VIT Vellore**.

The objective of this project is to build an end-to-end machine learning pipeline capable of predicting residential house prices using multiple regression algorithms.

The project begins with **Simple Linear Regression** as the baseline model and extends to advanced regression models such as Ridge, Lasso, Elastic Net, Decision Tree, Random Forest, Gradient Boosting, and other comparative approaches.

The project follows the complete Machine Learning lifecycle including:

- Business Understanding
- Data Collection
- Data Preprocessing
- Exploratory Data Analysis
- Feature Engineering
- Model Development
- Model Evaluation
- Hyperparameter Tuning
- Residual Diagnostics
- Result Interpretation
- Model Comparison
- Technical Documentation
